WPB Instagram Gallery Slider and Grid is highly customizable gallery plugin for WordPress.

== Description ==

WPB Instagram Gallery Slider and Grid is highly customizable gallery plugin for WordPress. It support Instagram images slider or grid with lightbox popup.
Adding Instagram image gallery in WordPress site is super easy using this plugin. Just connect the plugin with your Instagram account and you are ready to go.

It's a very easy to use plugin. You can show the Instagram image gallery by using our ShortCode. ShortCode comes with so many parameters, by using this parameters you can customize the gallery.

== Installation ==

* Install it as a regular WordPress plugin

* After install the plugin make sure you configure your Instagram Access Token with this plugin.

* Use this pluginâ€™s ShortCode to show the Instagram slider or grid in your site.

Method 1 (Upload):

1. Go to your Wordpress admin then go to Plugins > Add new > Upload

2. Then select the installable file fom you download folder.

3. Upload and active the plugin and you are ready to go.

Method 2 (FTP):

1. Upload the plugin to your server plugin directory and make sure you don't upload the zip file.

2. Then go back to your Wordpress admin and go to Plugins > Installed Plugins

3. If you successfully upload the plugin You will find WPB Instagram Gallery Slider and Grid here.

4. Active the plugin and you are ready to go.
